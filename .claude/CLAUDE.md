# Claude Code Instructions

> **Note**: This file contains project-specific instructions for Claude Code. Update this with your actual project details.

## Project Context

This is the **Claude Code Template** - a starter repository for building projects with AI assistance.

### Purpose of This Template
- Demonstrates best practices for Claude Code projects
- Provides reusable skills for common development tasks
- Shows how to structure projects for optimal AI collaboration
- Serves as a starting point for new projects

## Project Structure

### Important Files
- **PLAN.md**: Comprehensive project plan template (see this for architecture and requirements)
- **skills/**: Contains reusable skills for code review, testing, and documentation
- **SKILL_TEMPLATE.md**: Template for creating new skills
- **docs/**: Documentation including skills guide and examples

### When Working on This Project

1. **Reference PLAN.md** for:
   - Project architecture and design decisions
   - Development phases and current priorities
   - Coding standards and conventions
   - API design patterns
   - Security and performance requirements

2. **Use Available Skills**:
   - `code-review`: For comprehensive code reviews
   - `testing`: For generating test cases
   - `documentation`: For creating technical documentation

3. **Follow Best Practices**:
   - Keep examples clear and educational
   - Maintain consistency across documentation
   - Ensure all templates are easy to customize
   - Test that skills work as expected

## Instructions for Users of This Template

When someone uses this template, they should:

1. **Replace this CLAUDE.md** with their own project-specific instructions
2. **Customize PLAN.md** with their actual project plan
3. **Review and modify skills** to match their workflow
4. **Update documentation** to reflect their project

## Development Guidelines for This Template

- All example code should be simple and illustrative
- Documentation should be comprehensive but scannable
- Skills should be production-ready and well-commented
- Templates should include helpful inline guidance
